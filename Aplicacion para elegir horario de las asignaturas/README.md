#Aplicación para elegir horario de las asignaturas

 Aplicación desarrollada en modo escritorio (https://github.com/daster1324/enroll)
 
 Aplicación desarrollada en vista web (https://github.com/frmjar/enroll-web)


 Aplicación de escritorio desarrollada en java y que sigue una implementación basada en el modelo vista controlador (mvc). 
 Esta aplicación nos permite, de una manera más amigable, realizar nuestro horario del siguiente curso en el que nos vamos a
 matricular, ya que no tendremos que preocuparnos de ir anotando el horario de cada curso y grupo para compararlo con asignaturas 
 de otros cursos y grupos y saber si nos solapa, puesto que este trabajo lo hace la aplicación por nosotros. 
 
 El alumno tendrá que elegir el itinerario al que pertenece. el curso y el grupo del que desea ver las asignaturas. Si hay 
 alguna asignatura de las mostradas que le interese, la seleccionará para que forme parte de su horario (también tendrá la opción 
 de eliminarla). Tras esto, podrá seguir viendo otros cursos y grupos y los solapamientos se producen con la asignatura 
 previamente seleccionada, por lo que si alguna le interesa más, podrá cancelar la selección de la asignatura que no le interese 
 y dejará de formar parte de su horario. 
 
 Una vez terminada la selección de asignaturas, el usuario puede exportarla en formato csv. 
 
 Tambien dispone de version web.
